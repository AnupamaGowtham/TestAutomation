Project Description : This project automates the given 6 test cases using selenium with java.

Developed selenium based testNG framework for the automation of 6 test cases using Java. 

Followed oops concepts and POM(Page Object model) in the automation of testcases

src -> main/java/Anupama/pageobjects/ -> HomePageObjects.java file contains all the required locators and helper methods.

src -> test/java -> Anupama/QETechAssessment -> HomePage.java file contains all the Test methods using test anotations

src -> test/java -> QETechAssessment/data/DataReader.java file contains class for reading data from the json file.

Used Maven repository to add all the required dependencies in eclipse IDE.
